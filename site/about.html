About this site

A public notebook for practical product engineering.

blog.vinividivici.top

I use this site to write about the things I am building in public: smart TV applications, media playback details, small automation tools, and developer workflows that make repeated work easier.

The main thread right now is practical product engineering. The public work includes bili-webos for LG webOS TVs, claude-code-review for model-assisted review workflows, scenic-roads for map-based weekend route discovery, and small supporting tools such as webos-tv-app-skill and wechat-cli.

I prefer software that stays understandable: fewer background services, fewer runtime dependencies, and enough documentation that a future maintainer can pick up the project without reverse engineering every decision.
